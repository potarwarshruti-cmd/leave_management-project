# 🎓 Student Leave Management System

A web-based Student Leave Management System designed for ADYPU students and administrators.  
The system allows students to apply for leave digitally and enables administrators to manage, approve, or reject leave applications efficiently.

## 📌 Project Overview

Managing student leave requests manually can be time-consuming. This project provides a simple digital platform where:

- Students can create profiles
- Submit leave applications
- Upload medical certificates
- Track leave approval status
- Receive notifications
- Admin can manage all leave requests

---

## ✨ Features

### 👨‍🎓 Student Module

✔ Student login using college email  
✔ Create student profile  
✔ Store student details:
- Name
- URN
- Academic year
- Division
- Contact details
- ABC ID
- Profile picture

✔ Apply for leave  
✔ Upload medical certificate  
✔ View leave status  
✔ Receive approval/rejection notifications  

---

### 👨‍💼 Admin Module

✔ Admin dashboard  
✔ View all leave applications  
✔ Approve leave requests  
✔ Reject leave requests  
✔ View uploaded certificates  
✔ Track registered students  
✔ Export leave records as CSV
